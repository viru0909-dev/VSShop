# Product Images

Place your high-quality product images here.

## Naming Convention

Use descriptive names matching your products:
- `tomatoes.jpg` or `tomatoes.png`
- `milk.jpg`
- `apples.jpg`
- `chocolate-cake.jpg`
- `potato-chips.jpg`
- etc.

## Recommended Specifications

- **Format**: JPG or PNG
- **Size**: 800x800px minimum (square format works best)
- **Quality**: High resolution for crisp display
- **File Size**: Optimize to < 500KB each for fast loading

## Usage

Images will be imported in components like:
```javascript
import tomatoesImg from '../assets/products/tomatoes.jpg';
```

Or dynamically loaded based on product data from the backend.
